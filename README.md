### Vue Part
#### 1、nodejs官网下载安装node
 - 验证安装成功：node -v、npm -v
#### 2、安装yarn
 - npm install -g yarn
#### 3、使用vue脚手架创建vue环境
 - npm install -g vue-cli
#### 4、进入项目目录，初始化Vue
 - vue init webpack front_part(前端目录名)
#### 5、安装ElementUI
 - npm install element-ui -save
